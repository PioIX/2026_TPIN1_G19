# 2026_TPIN1_G19
### Esta página tratará de un juego llamado "Adivina el jugador". Tendrás un límite de 10 preguntas y 3 vidas donde preguntarás cuál es la liga, el país, el club y la posición del jugador que buscarás adivinar.
#### Las ligas serán la liga argentina, la liga francesa, la liga italiana, la liga española y la liga inglesa.
