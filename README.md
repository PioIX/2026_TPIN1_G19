# 2026_TPIN1_G19

Esta página tratará de un juego llamado **"Adivina el jugador"**. Tendrás un límite de 10 preguntas y 3 vidas donde preguntarás cuál es la liga, el país, el club y la posición del jugador que buscarás adivinar. Hay 100 jugadores cargados divididos en las ligas elegidas. Habrá ligas y equipos cargados en la base de datos que no servirán de nada ya que no hay jugadores para esas ligas, por lo que te gastarán preguntas si preguntas por ellas.

## Ligas

- Bundesliga
- Brasileirao
- Liga Argentina
- Liga Francesa
- Liga Italiana
- Liga Española
- Liga Inglesa
